# Swiggy Data Analysis Project

## Project Overview
This project analyzes Swiggy food delivery data to understand sales patterns, customer ratings, and ordering trends.

## Objectives
- Analyze weekly sales trend
- Analyze quarterly performance
- Compare veg vs non-veg sales
- Identify important business insights from food delivery data

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel
- Jupyter Notebook / PyCharm

## Project Structure
- `Data/Raw/` → raw dataset
- `Notebook/` → analysis notebook
- `README.md` → project documentation

## Key Analysis
- Weekly Trend Analysis
- Quarterly Performance Summary
- Food Type Sales Comparison
- Revenue-based insights

## Outcome
This project helped in practicing data cleaning, grouping, aggregation, and visualization using Python.
